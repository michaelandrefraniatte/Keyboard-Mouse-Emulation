The mouse moves to 0,0 instead of 500,500 because the executed command is literally "xdotool mousemove p p".

You need to construct your string first, e.g.:

char str[256];
int p1 = 500;
int p2 = 500;
sprintf(str, "xdotool mousemove %d %d", p1, p2);
system(str);
